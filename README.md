# Event Registration System (Backend) 

A robust, secure, and scalable backend application built for managing event registrations. This project was developed as part of my **Backend Developer Internship at CodeAlpha**. It provides full control over event creation via the Django Admin Panel and exposes clean RESTful APIs for viewing events and registering users.

## 🛠️ Tech Stack & Features

- **Backend Framework:** Django (Python)
- **Database:** SQLite (Lightweight & efficient relational database)
- **API Architecture:** RESTful APIs for seamless integration
- **Features:**
  - ✅ **Admin Panel:** Secure interface for administrators to create, update, and manage events.
  - ✅ **View Events API:** Open endpoint to fetch the list of all upcoming events in clean JSON format.
  - ✅ **User Registration API:** Secure endpoint allowing users to register for specific events with data validation.

## 🚀 Key Learning Outcomes
- Designing relational database schemas using Django Models (`Events` and `Registrations`).
- Working with Django's built-in Admin interface and customization.
- Implementing REST API concepts for data communication.
